# Каталог пользователей

Приложение для просмотра и поиска пользователей. Данные загружаются из публичного API [DummyJSON](https://dummyjson.com/).

## Стек

- React 19 + TypeScript
- Vite
- CSS Modules

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173) в браузере.

## Сборка

```bash
npm run build
```
